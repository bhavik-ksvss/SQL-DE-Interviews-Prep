# Day 1 Notes

## NULL Basics
- NULL = unknown
- NULL comparisons → UNKNOWN → not returned

## Must Use
IS NULL
IS NOT NULL
COALESCE(col, value)

## COUNT
COUNT(col) → ignores NULL
COUNT(*) → counts all rows

## Rules
Always handle NULL in filters

## Common Mistakes
- = NULL (wrong)
- Missing NULL in WHERE
- Using COUNT incorrectly

## Interview Tip
Mention NULL handling explicitly
